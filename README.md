# 🚀 Formação iOS

Bem-vindo à **Formação iOS**, uma trilha estruturada para transformar iniciantes em desenvolvedores iOS preparados para o mercado.

Esta formação não é apenas um conjunto de aulas.  
Ela é uma jornada prática baseada em projetos reais e progressão técnica.

---

# 🎯 Objetivo da Formação

Capacitar o aluno a:

- Criar aplicativos iOS do zero
- Trabalhar com múltiplas telas e navegação
- Manipular listas e dados dinâmicos
- Persistir dados localmente
- Consumir APIs reais
- Aplicar arquiteturas modernas
- Escrever testes automatizados
- Utilizar injeção de dependência
- Construir um projeto final profissional

Ao final, o aluno estará preparado para sua primeira vaga como **Desenvolvedor iOS Júnior**.

---

# 🧱 Estrutura da Formação

A formação é dividida em 4 níveis, organizados em 8 grandes desafios.
Cada nível representa uma evolução técnica real.

---

- 🟢 Nível 1 — Fundamentos iOS & Listas
- 🟡 Nível 2 — Persistência Local & Consumo de APIs
- 🔵 Nível 3 — Arquitetura & Testes
